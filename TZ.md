# Технічне завдання: Проект SHTORA

**Дата створення:** 29.12.2025  
**Версія:** 1.0 (чернетка для узгодження)

---

## 1. Мета і короткий опис проекту

**Проект:** SHTORA — сучасний сайт (каталог/інтернет-магазин) з можливістю замовлення та онлайн-оплати.

**Технології:**
- **Фронтенд:** Next.js / React
- **Бекенд/CMS:** Node.js, Directus CMS
- **База даних:** PostgreSQL / MySQL
- **Платежі:** LiqPay
- **Деплой:** Railway / Vercel / інше

**Цілі проекту:**
- Зручний каталог товарів/послуг
- Просте оформлення замовлення
- Безпечні онлайн-платежі
- Базове SEO для пошукових систем
- Інтеграція аналітики
- Хороша швидкодія і доступність

## 5. Приймальні критерії

### Каталог
- [ ] Фільтри працюють без перезавантаження сторінки
- [ ] Сторінки категорій/товарів індексуються пошуковиками

### Кошик
- [ ] Зберігається між перезавантаженнями браузера
- [ ] Коректно рахує знижки/доставку/суму

### Оплата
- [ ] Успішний платіж змінює статус замовлення на "оплачене"
- [ ] Дублювання не відбувається при повторному вебхуку
- [ ] Відмова/скасування відображаються правильно

### Емейли
- [ ] Відправляються з ретраями
- [ ] Логуються помилки
- [ ] Не дублюються

### SEO
- [ ] Є sitemap.xml, robots.txt, канонікали, OG, schema.org
- [ ] Lighthouse SEO ≥ 90 на головних сторінках

### Продуктивність/Доступність
- [ ] LCP/CLS в межах цілей
- [ ] Lighthouse Accessibility ≥ 90

### Адмінка
- [ ] Ролі/права працюють коректно
- [ ] Контент редагується без залучення розробника

## 9. Уточнювальні питання до замовника

Будь ласка, надайте відповіді на наступні питання для фіналізації ТЗ:

1. **Тип магазину:** тільки прийом заявок чи повна оплата онлайн?
2. **Доставка:** самовивіз, Nova Poshta, кур'єр? Чи потрібен розрахунок вартості?
3. **Податки/рахунки:** чи потрібні накладні/рахунки, ПДВ?
4. **Мови:** українська + які ще?
5. **Каталог:** приблизна кількість категорій/товарів, які атрибути/фільтри?
6. **Повернення коштів:** чи потрібні часткові/повні рефанди через LiqPay?
7. **Інтеграції:** CRM/ERP/Nova Poshta/Telegram-сповіщення?
8. **SEO:** чи є ключові семантичні ядра/метадані, редиректи зі старого сайту?
9. **Правові сторінки:** Політика конфіденційності, Публічна оферта, Умови повернення?
10. **Хостинг/домен/аналітика:** чи є готові акаунти, чи потрібна допомога у налаштуванні?
